### Discovered bugs (all documented in pitfalls.md)
1. **PyMuPDF `delete_annot` ไม่ persist** — use `doc.xref_set_key(page.xref, "Annots", "[]")` instead
2. **`a = doc[pi].first_annot` invalidates** — keep `page = doc[pi]` ref alive
3. **GDrive sync timeout on `shutil.copy2`** — use `os.open(O_WRONLY|O_TRUNC) + os.write()` for cloud-cached files
